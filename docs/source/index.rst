N-fold-edge Marker locator and tracker
======================================

N-fold-edge is a marker locator and tracker. It can locate the following types of markers with high accuracy and speed.

.. image:: _static/markers.png

Installation
------------

N-fold-edge is a python package and can be installed with pip, uv etc.

.. code-block:: shell

   pip install n-fold-edge

See :doc:`Installation <installation>` for more advanced methods of installation.

Markers
-------

Markers can be generated from the latex files in the patterns folder in the repository. A set of markers can also be downloaded from :download:`here <_static/patterns.pdf>`.
